# Sistema de Checkout

[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)

> Sistema de checkout completo com TypeScript. Carrinho de compras, cálculo de frete, pagamento e finalização de pedido.

🔗 **[GitHub](https://github.com/belmontprogramador/checkout)**

## ✨ Funcionalidades

- Carrinho de compras dinâmico
- Cálculo de frete por CEP
- Resumo do pedido em tempo real
- Múltiplas formas de pagamento
- Validação de formulários
- Fluxo completo até confirmação

**Stack:** TypeScript, Checkout, E-commerce · **Autor:** Felipe Belmont
