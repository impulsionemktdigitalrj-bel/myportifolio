> API de gerenciamento de usuários com Node.js, Express, Prisma ORM e autenticação JWT. Base sólida para sistemas que precisam de cadastro, login e controle de acesso.

🔗 **[GitHub](https://github.com/belmontprogramador/crud-simples-users)**

## 🛠️ Stack

- **Runtime:** Node.js
- **Framework:** Express
- **ORM:** Prisma
- **Banco:** PostgreSQL / SQLite
- **Autenticação:** JWT (access + refresh tokens)
- **Validação:** Zod

## ✨ Funcionalidades

- CRUD completo de usuários
- Registro com hash de senha (bcrypt)
- Login com JWT
- Middleware de autenticação
- Validação de dados com Zod
- Migrations com Prisma
