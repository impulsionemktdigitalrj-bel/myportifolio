# Folha de Ponto Digital

[![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![React](https://img.shields.io/badge/React-20232A?style=flat&logo=react)](https://react.dev/)

> Sistema completo de registro de ponto eletrônico com front-end React e API back-end em JavaScript. Inclui autenticação, registro de batidas e relatórios.

🔗 **Front-end:** [GitHub](https://github.com/belmontprogramador/front-folha-ponto)
🔗 **Back-end:** [GitHub](https://github.com/belmontprogramador/back-folha-de-ponto)

## ✨ Funcionalidades

- Registro de ponto (entrada/saída/almoço)
- Dashboard com horas trabalhadas
- Relatórios mensais exportáveis
- Autenticação de funcionários
- Cálculo automático de horas extras

**Stack:** JavaScript, React, API REST, Ponto Eletrônico · **Autor:** Felipe Belmont
