# Jobs — Plataforma de Vagas

[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)
[![React](https://img.shields.io/badge/React-20232A?style=flat&logo=react)](https://react.dev/)

> Plataforma web de empregos com TypeScript e React. Busca inteligente, filtros avançados e sistema de candidaturas online.

🔗 **[GitHub](https://github.com/belmontprogramador/jobs)**

## ✨ Funcionalidades

- Listagem de vagas com busca e filtros
- Detalhes da vaga com descrição completa
- Formulário de candidatura
- Design responsivo
- Dados mockados (JSON) prontos para integração com API

**Stack:** TypeScript, React, Plataforma Web · **Autor:** Felipe Belmont
