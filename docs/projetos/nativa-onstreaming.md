## 🛠️ Stack

- **Mobile:** React Native, TypeScript
- **Streaming:** Áudio em tempo real
- **Build:** Android (APK/AAB), iOS (IPA)

## ✨ Funcionalidades

- 🎵 Streaming de áudio ao vivo 24h
- 📻 Background playback (áudio continua ao sair do app)
- 🎨 Interface intuitiva e moderna
- 📱 Suporte Android + iOS
- 🔔 Notificações push
