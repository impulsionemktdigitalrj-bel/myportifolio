# Portal Proteger

[![React](https://img.shields.io/badge/React-20232A?style=flat&logo=react)](https://react.dev/)
[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)
[![Next.js](https://img.shields.io/badge/Next.js-000000?style=flat&logo=nextdotjs)](https://nextjs.org/)

> Portal institucional com sistema de gestão de conteúdo. Design profissional focado em acessibilidade e usabilidade.

## 🚀 Acesse

🔗 **[portalproteger.online](https://portalproteger.online/)**

## 🛠️ Stack

- **Front-end:** React, TypeScript, Next.js
- **Estilização:** CSS Modules / Styled Components
- **Hospedagem:** Cloudflare Pages

## ✨ Funcionalidades

- Sistema de gestão de conteúdo (CMS)
- Design responsivo (mobile-first)
- Navegação acessível (WCAG 2.1)
- SEO otimizado com Next.js SSR/SSG
- Formulário de contato integrado

## 🏗️ Estrutura

```
portal-proteger/
├── src/
│   ├── app/          # App Router (Next.js 14+)
│   ├── components/   # Componentes reutilizáveis
│   └── styles/       # Estilos globais
├── public/           # Assets estáticos
└── content/          # Conteúdo do CMS
```
